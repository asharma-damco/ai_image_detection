# Communications — AI Image Detection

One file per event: stakeholder email, Teams message, meeting transcript,
steering call notes, internal delivery discussion, demo session.

## File naming convention
YYYY-MM-DD-[event-slug].md
Examples:
  2026-05-18-kickoff-call.md
  2026-05-22-poc-demo-notes.md
  2026-05-25-steering-call-q1.md

## What belongs here
Preserve communications that explain:
- Commitments made
- Risks raised or acknowledged
- Decisions or direction changes
- Scope changes (in or out)
- Feedback on deliverables

Do not duplicate every message. Capture the ones that matter.

## Format per file
# [EVENT TYPE] — [DATE] | [attendees or sender/receiver]
Source: [call / email / Teams / in-person]
CHG-ID: [if this was logged via /capture]

[2–4 sentences: what happened, what was decided or noted]

[bullet list]

[bullet list with owner and due date]
